# 📌 Spring Boot Projects Collection

This repository contains multiple practical implementations using **Spring Boot 3**, focusing on real-world features like email integration, OTP verification, CRUD APIs, and more.

---

## 🚀 Topics Covered

### 📧 Email & SMTP Configuration
- **Gmail SMTP Server Settings**
  - Host, Username, and Password configuration for projects

- **Send Email using Java Mail Sender (JMS)**
  - Basic email sending implementation

- **Send Email with Attachment**
  - Attach files using Java Mail Sender in Spring Boot

---

### 🔐 Authentication & Security
- **OTP Verification using Email**
  - Generate and verify OTP via email using SMTP

- **Forgot Password & Reset Password**
  - Secure password reset flow with email verification

- **Send OTP/SMS**
  - OTP sending using Spring Boot 3

---

### 🔄 REST API Development
- **Spring Boot REST API CRUD Example**
  - Full CRUD operations using:
    - Spring Data JPA  
    - MySQL  
    - Postman testing  

---

### 🧾 Utility Features
- **Generate QR Code**
  - Create QR codes using Spring Boot 3 REST API

---

## 🛠️ Tech Stack

- Java 17+
- Spring Boot 3
- Spring Data JPA
- MySQL
- Java Mail Sender (JMS)
- REST APIs
- Postman
